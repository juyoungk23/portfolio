At Wells Fargo (July 2023 – April 2026), I built the backend data infrastructure behind internal reporting dashboards. Most of the work was replacing legacy technical debt with cloud-native systems without breaking anything a bank depends on.

## Core Engineering

- **System migration:** Designed the migration pipeline that moved legacy systems from MongoDB to SQL with **zero data loss**, validated by 100% data integrity checks.
- **Performance:** Refactored bottlenecked Python ETL scripts, cutting data processing latency by **60%**.
- **Release management:** Ran monthly production deployment cycles, coordinating code integration across distributed teams.

## What it taught me

The stack is enterprise (Teradata, Informatica), but the discipline travels. Banking data doesn't tolerate "mostly correct." Three years of that makes defensive coding, validation, and observability automatic, and I bring those habits to everything I build now.

## Tech Stack

Python, SQL, Shell Scripting, GCP, Postgres, MongoDB.
